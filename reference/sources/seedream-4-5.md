## Key Specifications Verified
- **Resolution**: 1K to 4K, up to 4096×4096 pixels (verified across 4+ sources)
- **Speed**: 2K images in ~1.8 seconds (Seedream 4.0 baseline, verified across 2+ sources)
- **Input Modes**: Text-to-image, image editing, multi-image composition — unified framework (verified across 4+ sources)
- **Reference Images**: Up to 14 reference images for guided editing (from existing page; official docs confirm multi-image fusion)
- **Aspect Ratios**: 1:1, 4:3, 3:4, 16:9, 9:16, 21:9 and more (verified across 2+ sources)
- **Batch**: Up to 15 images per prompt (verified 1 source)
- **Unique 4.5 Features**: Pixel-level subject extraction, cinematic lighting/atmosphere, poster text-image layout, spatial/physical reasoning (official Chinese doc)
