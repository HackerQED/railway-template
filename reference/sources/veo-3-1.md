## Key Specifications Verified
- **Resolution**: 720p, 1080p, 4K (3840×2160) — 4K via Flow/API/Vertex AI only (verified across 4+ sources)
- **Duration**: 4, 6, or 8 seconds per generation; extendable to ~148s via Scene Extension (verified across 3+ sources)
- **Frame Rate**: 24fps standard, up to 60fps in 4K mode (verified across 2+ sources)
- **Aspect Ratios**: 16:9 landscape, 9:16 portrait (verified across 3+ sources)
- **Input Modes**: Text-to-video, image-to-video, first+last frame control, up to 3 reference images (verified across 3+ sources)
- **Audio**: Native audio-video generation — dialogue (lip-sync), SFX, ambient sound, 48kHz sampling rate (verified across 4+ sources)
- **Unique Features**: Ingredients-to-video, Scene Extension (chain up to 20x), outpainting, object insertion/removal, style transfer, SynthID watermarking
- **Speed**: Fast variant ~30% faster than standard
